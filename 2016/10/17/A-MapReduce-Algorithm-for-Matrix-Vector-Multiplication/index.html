假设:

  • Square Matrix \(\mathcal{A}\) has dimension \(I\times I\) with elements \(a(i,k)\) for \(0\leq i<I\) and \(0\leq k<I\).
  • Column Vector \(\mathcal{B}\) has dimension \(I\times 1\) with elements \(b(k,0)\) for \(0\leq k<I\).
